⚡ Wan2GP Desktop Launcher v2.0

Install, maintain, launch — that's it.

1 What it is

🔑
Wan2GP Desktop Launcher is a wrapper — it does NOT replace Wan2GP. All generation, models, scheduling, LoRAs run inside Wan2GP's own Gradio server in your browser. The launcher just handles: one-click install with GPU detection, environment management, hardware-tuned config, and a button that starts Wan2GP in a visible terminal window. Nothing more. The browser is the viewer.
🧹
v2.0 stripped out everything that duplicated the browser. No embedded webview, no output file sidebar, no metadata reader, no floating terminal, no drag-drop injection. The launcher is trim and focused.

6 How it works

🖥️ Desktop Launcher Electron — dashboard UI,
IPC, auto-updater
→
📜 Temp .bat script Written to %TEMP%,
opens in new cmd window
→
🐍 wgp.py Wan2GP's Gradio server
on localhost:7860
→
🌐 Your Browser Opens automatically
when server is ready
⏱️
Poll, don't spawn. The launcher doesn't track the child process. Every 8 seconds it checks if the port is still open. When the terminal window closes and the port goes down, it logs a message and cleans up. Zero GPU usage — Electron runs on SwiftShader (CPU).
