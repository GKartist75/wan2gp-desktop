⚡ Wan2GP Desktop Launcher

Complete user guide — install, manage, launch, and tune

1 What it is

🔑
Wan2GP Desktop Launcher is a wrapper — it does NOT replace Wan2GP. All generation, models, scheduling, and LoRAs run inside Wan2GP's own Gradio server. The launcher handles: one-click install with GPU detection, environment management, hardware-tuned config, multiple launch modes, and real-time monitoring.

⚡ Auto-Tune

🖥️
Hardware Card
Click Detect to scan GPU, CUDA toolkit, VRAM, attention kernel support. Results shown live.
📋
Recommendation Card
Suggests optimal wgp_config.json settings: profiles (video/image/audio) with editable dropdowns, Scaled Int8 quantization (best quality–VRAM balance), VAE Config set to Auto (lets Wan2GP decide when tiling is needed — higher presets save VRAM but may cause banding), and VRAM safety coefficient.
✅
Apply
One click writes the settings to wgp_config.json. Also accessible from the dashboard ⚡ Auto-Tune button.
Profile lookup table — Wan2GP uses 5 profiles (1–5) from its mmgp memory manager. The auto-tune picks one based on your VRAM and system RAM:
VRAM \ RAMhigh (≥64GB)mid (≥32GB)low (<32GB)
very_high (≥24GB)P1P1P1
high (≥16GB)P1P2P2
mid (≥10GB)P3P4P4
low (<10GB)P3P5P5
P1 — full models in VRAM, pinned memory ⇢ P5 — no pinned memory, tight budgets, all quantized. Profiles, Scaled Int8 quant, and VAE Auto can all be changed in the dropdowns before applying.
💡
Tip: The ⚡ Auto-Tune button on the main dashboard (action grid) opens the Manage panel directly on the Auto-Tune tab — no need to navigate there manually.

🔧 Key decisions

• BrowserView kept alive forever (no destroy/recreate — prevents blank 2nd launch)
• Manage panel detaches BrowserView + opaque backdrop (z-index can't overlay BrowserView)
• checkPort() tests TCP before spawning — if already running, just connects
• All launch args (--server-port, --server-name, --share) injected consistently across all modes
• Progress-forcing bootstrap ensures tqdm bars render in piped output

9 Maintenance

➕
Auto-install prereqs
Missing Git, Python, uv, or conda? Click Download & Install — the launcher handles it silently.

Runs git pull in the repo, then re-installs requirements.

Upgrade Packages
Upgrade PyTorch, attention kernels, or any pip package individually.

Type a pip name in the env card — installs into active env instantly.

Fresh clone + install. Backs up plugins/finetunes/config first.
🗑️
Unlink Env
Red text button in env card header. Removes environment directory.
💻
Desktop Shortcut
Creates a standalone .bat that activates the env, starts Wan2GP, and opens your browser.
⬆️
Self-Update
Checks GitHub for new launcher releases. Download + install in one click.
⚙️
Launch Args
Extra flags passed to wgp.py (--profile, --attention, --listen, --share, etc.)
⚡
GPU detection uses direct shell commands (nvidia-smi, powershell, system_profiler, lspci) — no fragile Python wrapper that breaks on unrelated venvs. Fast, reliable, cross-platform.
