1 What it is

🔑
Wan2GP Desktop Launcher is a wrapper — it does NOT replace Wan2GP. All generation, models, scheduling, LoRAs run inside Wan2GP's own Gradio server. The launcher handles: one-click install with GPU detection, environment management, hardware-tuned config, and multiple launch modes. The browser (or the built-in Desktop viewer) is where you work.
🖥️
New in v2.1.3+: Launch in Desktop — Wan2GP runs inside the launcher itself (Electron BrowserView, no blank pages). Full nav controls, zoom, popout, and a dockable floating terminal. The app never leaves your dashboard.

6 How it works

🖥️ Desktop Launcher Electron — dashboard UI,
IPC, auto-updater
→
🧩 BrowserView Renders Wan2GP in-app
(not <webview> — blank on E40)
→
✨ manifest.json stub Intercepts Gradio's PWA manifest
request → fixes 404 blank page
→
🎬 Wan2GP (Gradio) Full generation UI embedded
with nav & zoom

Browser / No-GPU launch

📜 Temp .bat script spawn() + detached + unref()
avoids Windows quoting bugs
→
🐍 wgp.py Wan2GP's Gradio server
on localhost:7860
→
🌐 Browser YOUR default (or Chrome with
--disable-gpu flags for no-GPU)

Floating Terminal

📐 5 dock positions Bottom / Left / Top / Right /
Floating (draggable)
→
📏 Resizable Drag the bottom edge
80px – 60% window height
→
🟢 Server status Green/red dot in header
+ follow mode + search + export
💡
Key architectural decisions. BrowserView is kept alive forever (no destroy/recreate — prevents blank 2nd launch). The Manage panel detaches the BrowserView + shows an opaque backdrop (BrowserView always composites above DOM, so z-index can't work). checkPort() tests TCP before spawning — if already running, just connects.

7 Maintenance

➕
Auto-install prereqs
Missing Git, Python, uv, or conda? Click Download & Install — the launcher handles it silently.
⬆️
Update Wan2GP
Runs git pull in the repo, then re-installs requirements.
📈
Upgrade Packages
Upgrade PyTorch, attention kernels, or any pip package individually.
➕
Install Package
Type a pip name in the env card — installs into active env instantly.
🔄
Reinstall
Fresh clone + install. Backs up plugins/finetunes/config first.
🗑️
Unlink Env
Red text button in env card header. Removes environment directory.
💻
Desktop Shortcut
Creates a standalone .bat that activates the env, starts Wan2GP, and opens your browser automatically.
⬆️
Self-Update
Checks GitHub for new launcher releases. Download + install in one click.
⚙️
Launch Args
Extra flags passed to wgp.py (--profile, --attention, --listen, etc.)
⚡
GPU detection uses direct shell commands (nvidia-smi, powershell, system_profiler, lspci) — no fragile Python -c wrapper that breaks on unrelated venvs. Faster, more reliable, cross-platform.
